# Custom Shopify Theme

This repository contains a custom Shopify theme built for portfolio purposes.

## Features
- Custom Hero section using schema settings
- Font Picker section using Shopify `font_picker`
- Custom Featured Collection section
- Real-time Free Shipping Bar using Shopify AJAX cart (`/cart.js`)
- Theme-check compliant and performance optimized

## Tech Stack
- Shopify Liquid
- Sections & Schema
- JavaScript
- Shopify AJAX Cart

## Demo
Live demo available via Shopify development store (linked in portfolio).
